#build guide  
完成させるにあたり、用意するもの  
*RP2040-ZERO  
*ケース（bullyを流用する際は加工が必要）  
*tadpole pin  
*SMDダイオード…45個  
*cherry MX スイッチソケット…45個  
*約6mmのネオジム磁石　10個（私はセリアで購入）
